## 🚀 方法 1：使用 Simulator 应用（最简单，推荐）

### 步骤 1：打开 Simulator 应用

**最简单的方式**：
1. 按 `⌘ + Space`（Command + 空格）打开 Spotlight
2. 输入 "**Simulator**"
3. 按 `Enter` 打开

或者从 Xcode：
- **Xcode → Open Developer Tool → Simulator**

### 步骤 2：创建 iOS 设备

1. 在 Simulator 窗口中，点击菜单：**File → New Simulator**
2. 在弹出的对话框中：
   - **Device Type**：选择 **"iPhone 15"**（推荐）
   - **OS Version**：选择你下载的 iOS 版本（如 iOS 17.2）
3. 点击 **Create**

### 步骤 3：设备会自动启动

创建后设备会自动启动，你会在 Simulator 窗口中看到 iPhone 界面。

## 🎯 如果 Simulator 打不开或报错

### 先修复服务（如果遇到连接错误）

运行修复脚本：

```bash
cd "/Users/niyutong/Desktop/AI聚合器开发方式Two"
./restart-simulator-service.sh
```

然后等待 5-10 秒，再打开 Simulator。

### 或者手动修复

在终端运行：

```bash
# 停止所有模拟器进程
killall Simulator
killall com.apple.CoreSimulator.CoreSimulatorService

# 等待 3 秒
sleep 3

# 重新打开 Simulator
open -a Simulator
```

## 🐛 常见问题

### Q: Simulator 打不开？

**A**: 运行修复脚本：
```bash
./restart-simulator-service.sh
```

### Q: 找不到 iOS 版本？

**A**: 确保在 Xcode 中已完全下载：
- Xcode → Settings → Platforms
- 检查 iOS 版本是否显示为"已安装"

### Q: 设备创建失败？

**A**: 
1. 关闭 Simulator
2. 运行修复脚本
3. 重新打开 Simulator
4. 再次尝试创建设备
